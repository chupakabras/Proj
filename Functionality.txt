Final program functionality

Program creates online gift shop where clients can buy and order different gifts.
Every gift is limited, program shows how many gifts can be bought and how many are left.
Online shop has two modes: for buying gifts and viewing Your reservations. 
When user "enters" the online shop program asks to select the mode.
Also, window with time simulation appears, so user could control time flow
If "Gift Shop" is selected, table with various gifts appears.
User can buy a gift by clicking on it, then calendar shows up and user has to select the date (later he can change it).
If "Your Reservations" is selected, table with user's orders appears.
By clicking on gift user can change reservation time, if gift is still valid.
